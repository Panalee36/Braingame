## ข้อกำหนดเบื้องต้น

### ระบบปฏิบัติการ
- ✅ Windows 10/11
- ✅ macOS 10.14+
- ✅ Linux (Ubuntu 18.04+)

### ซอฟต์แวร์ที่ต้องติดตั้ง
- **Node.js**: เวอร์ชัน 18.0 ขึ้นไป ([ดาวน์โหลด](https://nodejs.org/))
- **npm** หรือ **yarn**: มาพร้อมกับ Node.js
- **Git** (ตัวเลือก): สำหรับ Clone repository

### ตรวจสอบการติดตั้ง
เปิด Terminal/Command Prompt และรันคำสั่งต่อไปนี้:

```bash
node --version
npm --version
```

## การติดตั้ง

### ขั้นตอนที่ 1: ไปยังโฟลเดอร์โปรเจกต์
```bash
cd d:\เกมฝึกสมอง
```

### ขั้นตอนที่ 2: ติดตั้ง Dependencies
```bash
npm install
```

หรือถ้าใช้ yarn:
```bash
yarn install
```

**หมายเหตุ**: กระบวนการนี้อาจใช้เวลา 5-10 นาที ขึ้นอยู่กับความเร็วอินเทอร์เน็ต

### ขั้นตอนที่ 3: ตรวจสอบการติดตั้ง
```bash
npm list
```

ควรเห็น:
```
brain-training-games@1.0.0
├── react@18.2.0
├── react-dom@18.2.0
├── next@14.0.0
└── tailwindcss@3.3.0
... (อื่น ๆ)
```

## การรันโปรแกรม

### โหมด Development (สำหรับพัฒนา)

```bash
npm run dev
```

**ผลลัพธ์ที่คาดหวัง**:

```
> next dev

  ▲ Next.js 14.0.0
  - Local:        http://localhost:3000
  - Environments: .env.local

✓ Ready in 2.5s
```

### เปิด Browser
ไปที่ `http://localhost:3000` ในแท็บเบราว์เซอร์

### หยุดการรัน
กด `CTRL + C` ใน Terminal

### โหมด Production

#### ขั้นตอนที่ 1: สร้าง Build
```bash
npm run build
```

#### ขั้นตอนที่ 2: รันเซิร์ฟเวอร์
```bash
npm start
```

## วิธีใช้งาน

### สำหรับผู้ใช้ทั่วไป

#### 1️⃣ เข้าสู่หน้าแรก
- เมื่อเปิดแอป คุณจะเห็นหน้าแรกพร้อมปุ่มเลือกเกม

#### 2️⃣ สมัครสมาชิก
- คลิก "สมัครสมาชิก"
- กรอกข้อมูล: ชื่อผู้ใช้, อีเมล, รหัสผ่าน, อายุ
- คลิก "สมัครสมาชิก"

#### 3️⃣ เข้าสู่ระบบ
- กรอกอีเมลและรหัสผ่าน
- คลิก "เข้าสู่ระบบ"

#### 4️⃣ เลือกเกม
- เลือกเกมจากหน้าแรก (มี 5 เกมให้เลือก)
- คลิกที่การ์ดของเกม

#### 5️⃣ เลือกระดับความยาก
- เลือกระดับจาก 1-5
- ระดับ 1: ง่ายสุด
- ระดับ 5: ยากสุด

#### 6️⃣ เล่นเกม
- ทำตามคำแนะนำที่ปรากฏ
- ดูคะแนนและเวลาบนหน้าจอ

#### 7️⃣ ดูผลลัพธ์
- เมื่อจบเกม ระบบจะแสดงคะแนนสรุป
- คลิก "เล่นอีกครั้ง" หรือ "กลับหน้าแรก"

#### 8️⃣ ตรวจสอบโปรไฟล์
- คลิก "👤 โปรไฟล์ของฉัน"
- ดูสถิติและความก้าวหน้า

- **เวลา**: 300 วินาที
- **วิธี**: ฟังเสียง แล้วเลือกรูปสัตว์
- **คะแนน**: ตอบถูก +10 + (ระดับ x 2)

## แก้ไขปัญหา

### ❌ ปัญหา: "npm: command not found"

**สาเหตุ**: Node.js ไม่ได้ติดตั้ง

**วิธีแก้**:
1. ดาวน์โหลด Node.js จาก https://nodejs.org/
2. ติดตั้ง (เลือก LTS)
3. รีสตาร์ท Terminal

---

### ❌ ปัญหา: "Port 3000 already in use"

**สาเหตุ**: มีโปรแกรมอื่นใช้พอร์ต 3000

**วิธีแก้**:
```bash
npm run dev -- -p 3001
```
แล้วไปที่ `http://localhost:3001`

---

### ❌ ปัญหา: "Module not found"

**สาเหตุ**: Dependencies ไม่ได้ติดตั้ง

**วิธีแก้**:
```bash
rm -rf node_modules package-lock.json
npm install
```

---

### ❌ ปัญหา: "Build failed"

**วิธีแก้**:
```bash
npm run lint        # ตรวจสอบ errors
npm run build       # สร้าง build ใหม่
```
